# ⚡ Electric Highway

A modern and responsive landing page for an electric vehicle highway concept. The project highlights clean UI design, smooth navigation, and an engaging user experience while showcasing the future of sustainable transportation.

## 🌐 Live Demo

**Website:** https://eyadwebdeveloper.github.io/electric-highway/

## 📌 Overview

Electric Highway is a front-end web project designed to present the concept of an electric highway in a visually appealing way. The website focuses on responsive layouts, intuitive navigation, and modern design principles to provide users with an enjoyable browsing experience across all devices.

## ✨ Features

* Modern and clean user interface
* Fully responsive design
* Mobile-friendly layout
* Smooth scrolling navigation
* Well-structured page sections
* Fast loading performance
* Cross-browser compatibility
* Organized and maintainable code

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/eyadwebdeveloper/electric-highway.git
```

Navigate to the project folder:

```bash
cd electric-highway
```

Open `index.html` in your browser or use a Live Server extension.
